# Citation Verification Checklist: VLA Module

## Overview
This document provides a comprehensive checklist to verify that all citations in the VLA module follow APA 7th edition style requirements with technical content adaptations.

## Verification Process

### Step 1: Citation Format Verification
For each citation in the VLA module, verify:

#### Journal Articles
- [ ] Author, A. A., Author, B. B., & Author, C. C. (Year). Title of article. Title of Periodical, volume(issue), pages. URL or DOI
- [ ] Proper capitalization (sentence case for article titles, title case for journal names)
- [ ] Correct punctuation and spacing
- [ ] Valid DOI or URL

#### Conference Proceedings
- [ ] Author, A. A., & Author, B. B. (Year, Month). Title of paper. In Conference Name (pp. page-page). Publisher. URL or DOI
- [ ] Proper capitalization and formatting
- [ ] Accurate page numbers and publisher information

#### Technical Reports and Preprints
- [ ] Author, A. A., Author, B. B., & Author, C. C. (Year). Title of report (Report No. if applicable). Institution or Organization. URL
- [ ] Complete institutional information
- [ ] Valid URL or repository reference

#### Software and APIs
- [ ] Organization or Developer. (Year). Software name (Version number) [Computer software]. URL
- [ ] Proper bracket notation for software identification
- [ ] Current and accessible URL

#### Technical Documentation
- [ ] Organization. (Year). Title of documentation. URL
- [ ] Current and accessible URL
- [ ] Proper organization attribution

#### Books
- [ ] Author, A. A. (Year). Title of book (Edition if not first). Publisher.
- [ ] Proper capitalization (sentence case for book titles)
- [ ] Complete publisher information

### Step 2: In-Text Citation Verification
Verify that in-text citations match reference list entries:

- [ ] Parenthetical citations: (Smith, 2023) for single author
- [ ] Parenthetical citations: (Smith & Johnson, 2023) for two authors
- [ ] Parenthetical citations: (Smith et al., 2023) for three or more authors
- [ ] Narrative citations: Smith (2023) showed that...
- [ ] Multiple citations: (Smith, 2023; Johnson et al., 2022)

### Step 3: Reference List Verification
Check the reference list format:

- [ ] Alphabetical order by first author's surname
- [ ] Hanging indent (first line flush left, subsequent lines indented 0.5 inches)
- [ ] Complete and accurate entries
- [ ] Consistent formatting throughout

### Step 4: Technical Content Specific Verification
For technical citations, verify:

- [ ] Code repositories properly cited when applicable
- [ ] Dataset citations follow proper format
- [ ] Preprint servers (e.g., arXiv) properly referenced
- [ ] Technical blog posts properly formatted
- [ ] Version numbers included for software and APIs

### Step 5: Authority and Currency Verification
- [ ] All sources are authoritative and relevant to technical content
- [ ] Sources are current (within 3-5 years for rapidly evolving fields)
- [ ] Peer-reviewed sources prioritized for technical claims
- [ ] Official documentation used for technology specifications

### Step 6: Accessibility Verification
- [ ] All URLs are accessible and functional
- [ ] DOI links resolve properly
- [ ] Archived versions available for critical sources
- [ ] Institutional access requirements noted where applicable
