# Component Documentation

This document provides an overview of the custom components available in the Physical AI & Humanoid Robotics book UI.

## Navigation Components

### Navbar
The main navigation bar component that includes dark mode toggle and mobile menu.

### Sidebar
A collapsible sidebar that displays the table of contents with modules and chapters.

### Breadcrumbs
Provides navigation context by showing the current page's location within the site hierarchy.

## Content Components

### BookLayout
A layout component that provides the two-column design with sticky sidebar for desktop and responsive layout for mobile.

### ProgressIndicator
Displays the user's progress through a chapter or module with a visual progress bar.

### SectionAnchor
Provides anchor links for sections within content, allowing direct linking to specific parts of a page.

## Interactive Components

### ToggleButton
A customizable toggle button component for enabling/disabling features.

### GlossaryTooltip
A tooltip component that provides definitions for technical terms when hovered or clicked.

### CollapsibleSection
A section that can be expanded or collapsed to show/hide content.

### CodeBlock
A code block component with syntax highlighting and copy functionality.

### CalloutBox
Styled boxes for notes, warnings, and concepts with appropriate icons and colors.

## Module and Assessment Components

### ModuleOverview
Displays an overview of a module including learning outcomes, timeline, and requirements.

### HardwareRequirementsCard
Displays hardware or software requirements in a card format.

### AssessmentChecklist
Interactive checklist for assessments with progress tracking.

### SubmissionCriteriaCard
Displays criteria for submissions and evaluation methods.

### CapstoneLayout
Layout component for capstone projects with step-by-step organization.

## Utility Components

### VisualSystemDiagram
Placeholder component for system diagrams with appropriate styling.

### LoadingSpinner
A loading spinner component for indicating loading states.

### WeeklyTimeline
Displays a timeline of weekly topics and content.
