# Glossary

## A

**AI Agent**: An autonomous system that perceives its environment and takes actions to achieve goals.

## D

**Digital Twin**: A virtual representation of a physical system that can be used for simulation and analysis.

## H

**Humanoid Robot**: A robot with human-like characteristics and capabilities.

## I

**Intelligent System**: A system that can learn, adapt, and make decisions based on data and context.

## R

**ROS 2 (Robot Operating System 2)**: A flexible framework for writing robot software that provides hardware abstraction, device drivers, libraries, and more.

## V

**Vision-Language-Action (VLA)**: A paradigm that integrates visual perception, language understanding, and physical action execution.
