# Module 3, Chapter 4: Synthetic Data Generation for Perception - Exercises

## Exercise 4.1: Analyzing Real-World Data Collection Limitations

**Objective**: Evaluate the practical limitations of real-world data collection for robotics perception.

**Instructions**:
1. Identify a specific perception task (e.g., pedestrian detection, object recognition)
2. List at least 5 practical limitations you would encounter when collecting real-world data for this task
3. For each limitation, propose how synthetic data generation could address it
4. Compare the cost and time requirements for real-world vs. synthetic data collection

**Deliverables**:
- Limitations analysis table
- 300-word comparison of real-world vs. synthetic approaches
- Cost-benefit analysis chart
