# Content Validation Tools

This directory contains tools for validating content quality, including Markdown formatting and APA citation standards.

## Markdown Validation

- Ensure all content follows Docusaurus-compatible Markdown format
- Validate proper heading hierarchy (h1, h2, h3, etc.)
- Check for proper code block formatting
- Verify image references are correct

## APA Citation Validation

- Verify citations follow APA 7th edition format
- Check that all cited sources are properly referenced in the text
- Ensure reference lists are properly formatted

## Reading Level Validation

- Content should maintain grade 10-12 reading level (Flesch-Kincaid)
- Use tools like Hemingway Editor or similar to assess readability
